Praticando com o Git e Github

O que é o GIT?

O git é um aplicativo SVC sistema de controle de versoes onde o programador usa para salvar versoes de um aplicativo ou voltar a uma versao antiga

O que é o Github

É um site liberado para todos mais com finalidade atrair os programadores para postarem seus scripts ou procurar outros scripts com finalidade ajudar os programadores
